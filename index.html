What is CI/CD?

CI/CD is a software development practice that helps developers automatically test, build, and deploy applications whenever changes are made to the code.

CI = Continuous Integration

Continuous Integration means developers regularly push their code changes to a shared repository such as GitHub.

CD = Continuous Delivery or Continuous Deployment

Continuous Deployment automatically deploys new changes to the live application after code is pushed to GitHub.
